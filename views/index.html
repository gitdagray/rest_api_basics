Introduction

API Details:

Accept Parameters

The API will respond to the following 'Accept:' values with appropriate content.

  • text/plain - Content will be returned as a plain string.
  • application/json - Content will be returned as a JSON object.
  • text/html - Content will be returned as an HTML page containing key:value pairs in an unordered list.

Operations

Method Path Description
GET / This page. HTML response.
GET /form Will return an example HTML5 form that sends requests to this API. HTML response.
GET /routes/json Will return example JSON data in the form of {"message": data}. JSON response.
GET /routes/time Will return the current date and time in JSON {"time": datetime}. JSON response.
GET /:word/echo Will return JSON that echoes the :word parameter {"echo": word}. JSON response.
GET, POST /routes/name?first=param&last=param Will return the first and last name parameters in requested response format.
GET /routes/name/about Will return a description of the /routes/name endpoint.
GET, POST, PUT /routes/echo-all?key1=param&key2=param Will return all parameters in requested response format.
